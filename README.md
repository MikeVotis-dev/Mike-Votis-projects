# Mike Votis Projects

A collection of Python projects showcasing interactive games, utilities, and graphics applications.

## 📁 Projects

### 1. 🎨 Tshirt_maker.py
A T-shirt design generator using Python's Turtle graphics library.
- Allows users to customize T-shirt colors
- Add custom text/logos to the design
- Interactive visual representation

**How to run:**
```bash
python Tshirt_maker.py
```

---

### 2. 💰 Should_I_buy_this.py
A smart budget calculator to help decide whether to purchase an item.
- Input product price and your budget
- Calculate monthly savings needed
- Get a timeline for when you can afford the item
- View leftover money after purchase

**How to run:**
```bash
python Should_I_buy_this.py
```

**Example:**
- Product: Laptop ($1000)
- Your Budget: ($300)
- Monthly Savings: ($100)
- Result: In 7 months you can buy it!

---

### 3. 🐢 turtle.catch.py
An interactive catching game using Python's Turtle graphics.
- Click on the moving turtle to catch it
- Complete 5 catches to win
- Fun and simple game mechanics

**How to run:**
```bash
python turtle.catch.py
```
